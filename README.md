# RN_Game_CandyCrush

### React native 로 match 3 알고리즘을 활용하여 Candy Crush 게임을 구현했습니다.

### 데모 영상 링크

https://youtu.be/GjM9r_QxJcQ

### 구현 사항

- [x] Progress 타이머
- [x] 잘못 된 이동 방지
- [x] 일치한 타일의 갯수에 비례한 점수 계산
- [x] 이동 횟수 계산
- [x] 이동 가능한 타일이 있는지 검사 후 없으면 타일 재배치
- [x] 초기 타일 배치 시 연속된 색깔의 타일이 배치되지 않도록

- [ ] 애니메이션 고도화
- [ ] 코드 리팩토링
- [x] 엄격한 타입 설정
- [x] 타일 연쇄 폭발 가능 하도록
